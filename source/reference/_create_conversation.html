Create a conversation

curl -XPOST -H "Content-type: application/json" -d '{
  "conversation": {
    "distinct_id": "",
    "properties": {}
  }
}' 'https://api.dialoganalytics.com/v1/b/:id/conversations?token=test'

Parameters

Param Description
distinct_id string Required

A unique conversation, thread or channel ID issued by the messaging platform.
properties hash Optional

Additional conversation properties.
